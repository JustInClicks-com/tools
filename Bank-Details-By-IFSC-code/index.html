“Bank details by IFSC" , this app will facilitate all the necessary bank details to the user. Users only need to input IFSC (Indian Financial System Code) and it will fetch all the data, showing bank details such as bank name, location, or a branch on the next page. To use this tool, open the app and enter the valid IFSC code and click on submit and you will receive the details of the bank showing:
  1. IFSC: Indian Financial System Code, is a unique code that every bank account holder have. If anyone wants to perform bank transactions then, the person has to provide their IFSC code.
  2. Bank Name:It is simply a bank name, the person has the account in.
  3. Bank Code:Bank code is assigned by the central bank to all the account holders, it is a 3 digit unique code.
  4. Branch:Bank branch is a physical location of the bank, containing the information where it is located
  5. Center:Bank centres is an area where indicating where the group of financial center lies.
  6. Contact:Contains contact information of the bank such as phone number, email id etc
  7. Address:Bank address carries the exact location of the bank in a particular area.
  8. City Provides the city name where the bank lies.
  9. District: Provides district name where the bank lies.
  10. State: Provides state name where the bank lies
  11. MICR:It is Magnetic Ink Character Recognition technology, a 9 digit code printed on the cheques.
  12. UPI:It is 4-6 digit pin use to perform bank transactions over money transfer apps
  13. RTGS: Real Time Gross Settlement is a way of fund transfer that allows the money transfer.
  14. NEFT:National Electronic Funds Transfer (NEFT) is a mode of online funds transfer
